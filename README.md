# chenJ-oa-parent-web

#### 介绍
云尚办公项目前端

#### 软件架构
vue-admin-template + Node.js + Pnpm + vue + ElementUI + Axios

#### 分支说明
 master-主分支
 develop-开发分支
 feature-功能分支
 hotfix-缺陷修改分支（基于develop或者release分支创建）
 release-发布分支（暂时不发布）

#### 安装教程
1.  下载npm
2.  安装pnpm
3.  pnpm install安装依赖  pnpm启动项目
